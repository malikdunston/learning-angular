## Starting a Peoject
-	install cli
	npm i -g @angular/cli
-	create new app...
	ng new [name of app]
	-- you can optionally add router... or add it manually later.
-	move to directory, start dev server...
	ng serve
	-- runs on localhost:4200... opens up the default landing page...
